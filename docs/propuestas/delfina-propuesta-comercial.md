# Propuesta Comercial: Delfina
## Asistente Inteligente de Viajes

---

## Resumen Ejecutivo

**Delfina** es un asistente conversacional de viajes impulsado por inteligencia artificial. Se integra como una burbuja flotante en su motor de búsqueda actual, ofreciendo a las 1.200 agencias de su red una herramienta de atención y planificación de viajes potenciada por IA.

---

## La Solución

### Integración

Una **burbuja flotante "Delfina"** se añade al motor de búsqueda existente. Al hacer clic, el usuario es redireccionado a nuestra plataforma donde interactúa con el asistente.

```
┌─────────────────────────────────────────────────────────────┐
│              SU MOTOR DE BÚSQUEDA ACTUAL                    │
│                                                             │

│                                          │   💬 Delfina  │  │
│                                          └───────────────┘  │
└─────────────────────────────────────────────────────────────┘
                              │
                              │ click
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                 PLATAFORMA DELFINA                          │

│  │  Chat conversacional con IA                         │   │
│  │  • Búsqueda de vuelos (conectado a Lleego)          │   │
│  │  • Planificación de itinerarios                     │   │
│  │  • Recomendaciones de destinos                      │   │
│  │  • Atención 24/7                                    │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### Capacidades

- **Búsqueda de vuelos en tiempo real** — Conectado a GDS, NDC y low-cost vía API Lleego
- **Planificación de viajes** — Itinerarios multi-ciudad con actividades reales
- **Multiidioma** — Español, inglés, portugués
- **Disponibilidad 24/7** — Sin intervención humana para consultas estándar

## Modelo Comercial

### Componentes

| Concepto | Descripción |
|----------|-------------|
| **Integración inicial** | Setup de la burbuja en su motor + conexión con Lleego |
| **Fee por agencia** | Cobro mensual por cada agencia activa que utilice Delfina |

### Requisitos del Cliente

1. **Cuenta en Lleego** — El cliente contrata directamente la API de vuelos y provee credenciales
2. **Acceso técnico** — Para insertar la burbuja en el motor de búsqueda existente
